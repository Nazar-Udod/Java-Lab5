Інструкція до використання:
- Переконайтеся, що Java встановлена на вашій системі
- Відкрийте термінал
- Скопіюйте репозиторій на свою систему за допомогою команди git clone або завантажте його іншим чином
- Перейдіть у необхідний каталог за допомогою команди cd [шлях до репозиторію]\src
- Скомпілюйте програму за допомогою команди javac Main.java
- Запустіть програму за допомогою команди java Main
- Програма дозволяє працювати зі списком тарифів мобільного оператора (програма сама створить початковий тестовий список тарифів)
- Програма запропонує список дій, виберіть дію введенням числа у консоль
- Програма у тому числі дозволяє редагувати типізовану колекцію тарифів (використовуйте тарифи з початкового списку)
- Програма виведе результат залежно від дії
